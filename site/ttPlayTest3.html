Play Test 3

Here we are, the final countdown, the final stand, the moment we have coded so hard for. The second play test was a nice jump and a big improvment from the first play test. We made a lot of progress and received a lot of great feedback from the class. The goal of this play test was to stop the "What's this?" and "Why's this?" from coming out of our players. So to do this we have to fix all the pointless bullets flying around, the memory leaks, and the lifeless character who slides on the ground, and make the game more intuitive and fun.
Less thinking, more playing.

Goals

  • Running Character Sprite
  • Graphic improvements
  • Arrow key controls
  • Better Score UI
  • AOE Time Manipulation
  • Fixing memory leaks
  • Adding shooting turrets
  • SQUASHING ALL THE BUGS
